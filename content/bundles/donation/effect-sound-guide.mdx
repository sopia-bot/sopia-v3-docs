# 효과음 가이드

효과음은 TTS가 재생되기 전에 먼저 재생되는 사운드입니다. 이 가이드에서는 효과음을 설정하고 커스터마이징하는 방법을 안내합니다.

## 효과음이란?

도네이션을 받았을 때 다음 순서로 재생됩니다:

```
1. 효과음 재생 ← 이 부분!
   ↓
2. TTS 음성 재생
```

효과음을 사용하면 방송이 더 역동적이고 재미있어집니다!

## 효과음 종류

### 1. 기본 효과음

모든 스티커에 공통으로 적용되는 효과음입니다.

- 스티커별 효과음이 없을 때 재생됩니다
- 번들에 기본으로 제공되는 효과음이 있습니다
- 직접 업로드하여 변경할 수 있습니다

### 2. 스티커별 효과음

특정 스티커에만 적용되는 효과음입니다.

- 스티커마다 다른 효과음을 설정할 수 있습니다
- 기본 효과음보다 우선 적용됩니다

## 효과음 페이지 접근

## 기본 효과음 설정하기

### 새로 업로드하기

1. **기본 효과음 설정하기** 버튼을 클릭합니다
2. 파일 선택 대화상자가 열립니다
3. 오디오 파일을 선택합니다
4. 미리듣기로 확인합니다
5. 볼륨을 조절합니다 (기본값: 50%)
6. **저장** 버튼을 클릭합니다

### 기존 효과음 사용하기

번들에 기본으로 제공되는 효과음(`default.mp3`)이 있습니다.
- 별도로 설정하지 않으면 이 효과음이 사용됩니다

## 스티커별 효과음 설정하기

### 새 효과음 추가하기

1. **새 효과음 추가하기** 버튼을 클릭합니다
2. **스티커 선택** 대화상자가 열립니다
3. 원하는 스티커를 선택합니다
4. 오디오 파일을 업로드합니다
5. 미리듣기로 확인합니다
6. 볼륨을 조절합니다
7. **저장** 버튼을 클릭합니다

### 효과음 수정하기

1. 효과음 목록에서 수정할 효과음을 찾습니다
2. 목록 아이템을 클릭하면 오디오 창이 열립니다.
3. 오디오 파일을 변경하거나 볼륨을 조절합니다
4. **저장** 버튼을 클릭합니다

### 효과음 삭제하기

1. 효과음 목록에서 삭제할 효과음을 찾습니다
2. **삭제** 버튼을 클릭합니다
3. 확인 대화상자에서 **확인**을 클릭합니다

## 오디오 파일 준비하기

### 지원하는 오디오 포맷

다음 오디오 포맷을 지원합니다:

- **MP3** (권장)
- **WAV**
- **OGG**
- **M4A**
- **FLAC**

### 오디오 파일 권장 사항

| 항목 | 권장 값 | 설명 |
|------|---------|------|
| **길이** | 1-5초 | 너무 길면 TTS 재생이 지연됩니다 |
| **비트레이트** | 128-320 kbps | 음질과 파일 크기의 균형 |
| **샘플레이트** | 44.1 kHz | 표준 오디오 샘플레이트 |
| **파일 크기** | 1 MB 이하 | 로딩 속도를 위해 |

### 오디오 편집 기능

오디오의 시작과 끝 부분을 자를 수 있습니다.

- **자르기**: 오디오의 시작과 끝 부분 자르기
