# 소피아 도네이션 번들

**소피아 도네이션 번들**은 스푼라디오에서 도네이션(스푼 선물)을 받았을 때 자동으로 TTS(Text-to-Speech)로 메시지를 읽어주고, 효과음을 재생하는 번들입니다.


## 주요 기능

### 1. 템플릿 시스템
도네이션 조건에 따라 다양한 음성과 설정을 자동으로 적용할 수 있습니다.

- **스티커 방식**: 특정 스티커를 받았을 때 지정한 음성으로 읽기
- **임계치 방식**: 특정 스푼 개수 이상을 받았을 때 지정한 음성으로 읽기
- **음성 선택**: ElevenLabs의 다양한 한국어 음성 중 선택 가능
- **볼륨 조절**: 템플릿별로 음성 볼륨 조절 가능 (0-100%)

### 2. 효과음 설정
스티커별로 개별 효과음을 설정할 수 있습니다.

- 기본 효과음 설정
- 스티커별 커스텀 효과음 설정
- 오디오 파일 업로드 및 편집
- 효과음 볼륨 조절
- 미리듣기 기능

### 3. 세부 설정
도네이션 처리 동작을 세밀하게 조정할 수 있습니다.

- 채팅 입력 대기 시간 설정
- 음성 및 효과음 재생 순서 설정

## 사용 흐름

1. 시청자가 스푼 선물을 보냅니다
2. **설정한 대기 시간 내에** 시청자가 채팅 메시지를 입력합니다
3. 효과음이 재생됩니다
4. TTS가 채팅 메시지를 읽어줍니다
