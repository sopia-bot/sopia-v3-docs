# 템플릿 가이드

템플릿은 도네이션 조건에 따라 자동으로 적용되는 음성 설정입니다. 이 가이드에서는 템플릿을 효과적으로 활용하는 방법을 안내합니다.

## 템플릿 구성 요소

템플릿은 다음 요소로 구성됩니다:

| 항목 | 설명 | 필수 여부 |
|------|------|-----------|
| **템플릿 이름** | 템플릿을 구분하는 이름 | 필수 |
| **동작 방식** | 스티커 또는 임계치 | 필수 |
| **조건** | 스티커 ID 또는 스푼 개수 | 필수 |
| **음성** | 사용할 음성 목록 | 필수 |
| **볼륨** | 음성 재생 볼륨 (0-100%) | 필수 (기본값: 70%) |

## 템플릿 생성하기

### 1. 스티커 방식 템플릿

특정 스티커를 받았을 때만 작동하는 템플릿입니다.

**사용 예시**:
- "하트 스티커는 A 목소리로"
- "폭죽 스티커는 B 목소리로"

**생성 방법**:

1. **새 템플릿 추가** 버튼 클릭
2. 템플릿 이름 입력 (예: "하트 스티커 전용")
3. **동작 방식**: "스티커" 선택
4. **스티커 선택** 버튼 클릭
5. 원하는 스티커를 목록에서 선택
6. 음성 선택 (여러 개 선택 가능)
7. 볼륨 조절
8. **저장** 클릭

### 2. 임계치 방식 템플릿

특정 스푼 개수 이상을 받았을 때 작동하는 템플릿입니다.

**사용 예시**:
- "100 스푼 이상은 일반 목소리로"
- "500 스푼 이상은 감사 인사 목소리로"
- "1000 스푼 이상은 특별한 목소리로"

**생성 방법**:

1. **새 템플릿 추가** 버튼 클릭
2. 템플릿 이름 입력 (예: "500 스푼 이상")
3. **동작 방식**: "임계치" 선택
4. **임계치** 입력 (예: 500)
5. 음성 선택
6. 볼륨 조절
7. **저장** 클릭

## 음성 선택하기

### 사용 가능한 음성

다양한 음성을 사용할 수 있습니다.

**음성 선택 팁**:
- 여러 음성을 선택하면 매번 랜덤으로 선택됩니다
- 다양한 음성을 사용하면 방송이 더 재미있어집니다
- 상황에 맞는 음성을 선택하세요

### 음성 미리듣기

우측 재생 버튼을 누르면 목소리에 대한 미리듣기를 할 수 있습니다.

## 템플릿 수정하기

1. 메인 화면에서 수정할 템플릿을 선택합니다
2. 설정을 변경합니다
3. **적용** 버튼을 클릭합니다

**주의**: 템플릿을 수정한 후에는 반드시 **적용** 버튼을 눌러야 변경 사항이 저장됩니다!

## 템플릿 삭제하기

1. 메인 화면에서 삭제할 템플릿 우측의 삭제 버튼을 클릭합니다.
2. **적용** 버튼을 클릭합니다.

## 여러 템플릿 관리하기

### 템플릿 매칭 규칙

여러 템플릿이 있을 때 다음 순서로 매칭됩니다:

1. **스티커 방식** 템플릿이 먼저 체크됩니다
   - 받은 스티커와 정확히 일치하는 템플릿을 찾습니다

2. **임계치 방식** 템플릿이 체크됩니다
   - 받은 스푼 개수가 임계치 이상인 템플릿을 찾습니다
   - 여러 임계치 템플릿이 조건을 만족하면 **가장 높은 임계치**를 사용합니다

### 템플릿 조합 예시

다음과 같이 여러 템플릿을 조합할 수 있습니다:

```
템플릿 1: 하트 스티커 (스티커 방식)
템플릿 2: 폭죽 스티커 (스티커 방식)
템플릿 3: 1000 스푼 이상 (임계치 방식)
템플릿 4: 500 스푼 이상 (임계치 방식)
템플릿 5: 100 스푼 이상 (임계치 방식)
```

**동작 예시**:

| 받은 도네이션 | 선택되는 템플릿 | 이유 |
|--------------|----------------|------|
| 하트 스티커 (200 스푼) | 템플릿 1 | 스티커 방식이 우선 |
| 폭죽 스티커 (1500 스푼) | 템플릿 2 | 스티커 방식이 우선 |
| 일반 스티커 (1200 스푼) | 템플릿 3 | 가장 높은 임계치 |
| 일반 스티커 (600 스푼) | 템플릿 4 | 가장 높은 임계치 |
| 일반 스티커 (150 스푼) | 템플릿 5 | 유일하게 만족하는 임계치 |
| 일반 스티커 (50 스푼) | 적용 안 됨 | 조건 불만족 |

## 볼륨 조절하기

### 볼륨 설정 범위

- **최소**: 0% (음소거)
- **최대**: 100% (최대 볼륨)
