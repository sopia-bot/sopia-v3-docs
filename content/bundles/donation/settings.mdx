# 설정 가이드

이 가이드에서는 도네이션 번들의 세부 설정을 조정하는 방법을 안내합니다.

## 설정 페이지 접근

1. 왼쪽 메뉴에서 **설정** 아이콘을 클릭합니다
2. 설정 페이지가 열립니다

## 채팅 입력 대기 시간

### 개요

채팅 입력 대기 시간은 스푼 선물을 받은 후 **시청자가 채팅을 입력할 때까지 기다리는 시간**입니다.

### 동작 방식

```
1. 시청자가 스푼 선물 전송
   ↓
2. 대기 시작 (설정한 시간 동안)
   ↓
3-1. 시청자가 채팅 입력 (대기 시간 내)
     → 효과음 + TTS 재생

3-2. 시청자가 채팅 미입력 (대기 시간 경과)
     → 재생 안 됨
```

### 설정 방법

1. 설정 페이지에서 **채팅 입력 대기 시간** 항목을 찾습니다
2. 슬라이더 또는 입력 필드로 시간을 조정합니다
3. **저장** 버튼을 클릭합니다
