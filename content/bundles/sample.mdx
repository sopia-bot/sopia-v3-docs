# 샘플 번들 문서

이 문서는 API 경로를 통해 동적으로 로드되는 MDX 콘텐츠의 예시입니다.

## 기능

동적 MDX 로드는 다음과 같은 이점이 있습니다:

- **실시간 데이터**: 문서를 빌드 시간이 아닌 요청 시점에 로드합니다.
- **외부 소스 통합**: 외부 API나 데이터베이스의 콘텐츠를 통합할 수 있습니다.
- **동적 콘텐츠**: 사용자 또는 다른 데이터에 기반한 동적 콘텐츠를 보여줄 수 있습니다.

## 코드 예제

```javascript
// 동적 데이터 로드 예시
async function loadDynamicData() {
  const response = await fetch('/api/data');
  const data = await response.json();
  return data;
}
```

## 결론

이 접근 방식을 사용하면 빌드 시간에 정적으로 정의된 문서 외에도 
동적으로 생성되거나 외부 소스에서 가져온 콘텐츠를 쉽게 표시할 수 있습니다.
