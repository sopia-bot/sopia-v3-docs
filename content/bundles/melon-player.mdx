# 멜론 플레이어

청취자들이 채팅으로 멜론 차트의 곡을 신청하고, DJ가 편리하게 재생할 수 있는 신청곡 시스템입니다.

<Callout type="error">

재생목록 반복이나 셔플은 꺼주세요.

이 내용을 지키지 않았을 시 신청곡 번들이 오작동 할 수 있습니다.

## 주요 기능

- 청취자가 채팅으로 곡 검색 및 신청
- 멜론 차트에서 랜덤 곡 추천
- 신청 횟수 제한으로 도배 방지
- 자동 재생 및 대기열 관리
- 웹 기반 관리 페이지

## 청취자용 명령어

### 신청곡 검색하기

채팅창에 `!신청곡 [검색어]`를 입력하면 곡을 검색할 수 있습니다.

```
!신청곡 아이유 블루밍
```

</Callout>

### 신청 횟수 제한

도배를 방지하기 위해 **1인당 신청 횟수가 제한**됩니다. 기본값은 3회입니다.

- 신청할 때마다 남은 횟수가 표시됩니다
- 제한에 도달하면 더 이상 신청할 수 없습니다
- DJ가 초기화하면 다시 신청할 수 있습니다

### 명령어 별칭

같은 기능을 하는 다양한 명령어를 사용할 수 있습니다.

| 기본 명령어 | 별칭 |
|------------|------|
| `!신청곡` | `!신청`, `!노래`, `!request`, `!req` |
| `!추천곡` | `!랜덤`, `!recommend`, `!random` |


### 신청 횟수 초기화

모든 청취자의 신청 횟수를 초기화합니다.

```
!신청곡 초기화
```

<Callout type="info">
신청 횟수 초기화는 방송 시작 시, 또는 신청이 저조할 때 사용하면 좋습니다.
</Callout>

### 대기열 초기화

신청곡 대기열을 모두 비웁니다.

```
!대기열 초기화
```

<Callout type="error">
⚠️ **주의:** 대기열 초기화는 되돌릴 수 없습니다. 모든 신청곡이 삭제됩니다.
</Callout>

### DJ 명령어 요약

| 명령어 | 설명 |
|--------|------|
| `!스킵` | 현재 곡 스킵 |
| `!신청곡 초기화` | 신청 횟수 초기화 |
| `!대기열 초기화` | 대기열 비우기 |

### 현재 재생 중

현재 멜론 플레이어에서 재생 중인 곡 정보가 표시됩니다.

- **앨범 아트** - 곡의 앨범 이미지
- **곡 정보** - 제목, 아티스트, 앨범명
- **진행 바** - 현재 재생 위치
- **재생 컨트롤** - 이전/재생·일시정지/다음 버튼

### 신청곡 대기열

신청된 곡들이 대기열에 표시됩니다.

| 항목 | 설명 |
|------|------|
| 앨범 아트 | 곡의 앨범 이미지 |
| 제목 | 곡 제목 |
| 아티스트 | 가수/그룹 이름 |
| 신청자 | 신청한 청취자 닉네임 |
| 상태 | 대기 중 / 재생 중 / 재생됨 |
| 액션 | 재생, 삭제 버튼 |

### 상태 표시

| 상태 | 설명 |
|------|------|
| 🟡 **대기 중** | 아직 재생되지 않은 곡 |
| 🟢 **재생 중** | 현재 재생 중인 곡 |
| ⚪ **재생됨** | 이미 재생이 완료된 곡 |

## 자동 재생

신청곡이 끝나면 자동으로 다음 곡이 재생됩니다.

- 대기열에 곡이 있으면 순서대로 자동 재생
- 곡이 끝나면 자동으로 "재생됨" 상태로 변경
- 대기열이 비면 재생이 멈춥니다

<Callout type="info">
멜론 플레이어에서 직접 다음 곡으로 넘기면, 대기열 상태도 자동으로 업데이트됩니다.
</Callout>
