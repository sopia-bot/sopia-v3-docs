# DJ 보드 에디터 사용 가이드

DJ 보드 에디터는 노션(Notion)과 비슷한 느낌의 강력한 콘텐츠 편집기입니다. <br/>
방송 소개, 일정 관리 등 DJ 활동에 필요한 모든 것을 한 곳에서 관리할 수 있습니다.

<Callout type="error">
DJ 보드 에디터는 현재 베타 버전입니다. 기능이 변경될 수 있습니다.<br/>
**파트너 DJ**만 사용할 수 있습니다.
</Callout>

2. 기본 2x2 표가 생성됩니다
3. 셀을 클릭하여 내용을 입력하세요

### 표 조작하기

## D-Day 카운트다운

중요한 날짜까지 남은 시간을 실시간으로 보여줍니다.

### D-Day 추가하기

1. `/dday` 입력
2. 제목을 입력하세요 (예: "첫 방송")
3. 목표 날짜를 선택하세요
4. 설명을 추가할 수 있습니다 (선택사항)

{/* TODO: D-Day 추가 다이얼로그 스크린샷 */}
![D-Day 추가](/bundles/dj-board/dday-1.png)

### 표시 방식

- **D-Day 숫자** - "D-7" (7일 남음) 또는 "D+3" (3일 지남)
- **실시간 카운트다운** - 시간, 분, 초까지 표시
- **상태별 색상**
  - 🔴 빨강 - 7일 이하 또는 지난 날짜
  - 🟠 주황 - 8~30일
  - 🟢 초록 - 31~90일
  - 🔵 파랑 - 91일 이상

## 데이터 테이블

엑셀처럼 데이터를 관리할 수 있는 고급 표입니다.

### 데이터 테이블 만들기

1. `/datatable` 입력
2. 테이블 이름을 정하세요
3. 필요한 열(컬럼)을 추가하세요

{/* TODO: 데이터 테이블 생성 다이얼로그 스크린샷 */}
![데이터 테이블 생성](/bundles/dj-board/data-table-1.png)

### 데이터 테이블 기능

- **정렬** - 열 제목을 클릭하면 오름차순/내림차순 정렬
- **검색** - 특정 데이터를 빠르게 찾기
- **필터링** - 조건에 맞는 데이터만 보기
- **페이지네이션** - 데이터가 많을 때 페이지로 나누어 표시

### 행/열 관리

- **행 추가** - "행 추가" 버튼 클릭
- **행 삭제** - 행의 체크박스 선택 후 "선택 항목 삭제"
- **열 관리** - 상단 메뉴에서 열 추가/삭제/이름 변경

## 페이지 관리

여러 페이지를 만들어 콘텐츠를 체계적으로 관리할 수 있습니다.

### 새 페이지 만들기

1. 에디터에서 `/페이지` 입력
2. 새 페이지 링크가 생성됩니다
3. 링크를 클릭하면 빈 페이지로 이동합니다

{/* TODO: 에디터에 페이지 링크가 삽입된 스크린샷 */}

<Callout type="info">
새 페이지를 만들면 자동으로 빈 페이지가 저장되어 404 오류가 발생하지 않습니다.
</Callout>

### 페이지 사이드바

왼쪽 사이드바에서 모든 페이지를 확인할 수 있습니다:

- **🏠 메인 페이지** - 가장 처음 보이는 페이지
- **📄 모든 페이지** - 생성한 모든 페이지 목록

### 페이지 이동하기

- 사이드바에서 페이지 이름을 클릭하세요
- 현재 페이지는 배경색으로 강조 표시됩니다
- 페이지 링크를 클릭해도 이동할 수 있습니다

### 페이지 링크 수정하기

1. 페이지 링크를 **더블클릭**하세요
2. 또는 링크에 마우스를 올리고 ✏️ 아이콘을 클릭하세요
3. 링크 텍스트를 변경할 수 있습니다
4. **"저장"** 버튼을 클릭하세요

{/* TODO: 페이지 링크 편집 다이얼로그 스크린샷 */}
![페이지 링크 편집](/bundles/dj-board/pagelink-2.png)

<Callout type="info">
링크 텍스트를 변경해도 실제 페이지 URL은 변경되지 않습니다. 
페이지의 제목은 첫 번째 제목(H1-H6)이 자동으로 사용됩니다.
</Callout>

### 페이지 삭제하기

1. 사이드바에서 페이지에 마우스를 올리세요
2. 오른쪽에 **X** 버튼이 나타납니다
3. X 버튼을 클릭하세요
4. 확인 창에서 **"삭제"**를 클릭하세요

{/* TODO: 페이지 삭제 확인 다이얼로그 스크린샷 */}
![페이지 삭제](/bundles/dj-board/pagelink-4.png)

<Callout type="error">
⚠️ **주의:** 페이지를 삭제하면 모든 내용이 영구적으로 삭제되며 되돌릴 수 없습니다!
</Callout>

## 유용한 팁

### 자동 저장

<Callout type="success">
모든 변경사항은 2초 후 자동으로 저장됩니다. 저장 버튼을 누를 필요가 없습니다!
</Callout>

### 페이지 제목 자동 설정

페이지에 제목(H1~H6)을 작성하면 자동으로 페이지 제목으로 사용됩니다. 
제목이 없으면 "Untitled"로 표시됩니다.
