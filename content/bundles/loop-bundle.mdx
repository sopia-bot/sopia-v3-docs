## 소개

반복 번들은 방송 중 일정 시간마다 특정 메시지를 자동으로 반복 전송하는 기능을 제공합니다.
공지사항, 이벤트 안내, 규칙 알림 등을 주기적으로 청취자에게 전달할 때 유용합니다.

{/* 이미지: 반복 번들 메인 화면 스크린샷 (설정 목록이 보이는 전체 UI) */}
![반복 번들](/bundles/loop-bundle/loop-1.png)

## 주요 기능

- **다중 반복 설정**: 여러 개의 반복 메시지를 동시에 등록하고 관리할 수 있습니다.
- **개별 제어**: 각 반복을 개별적으로 시작/중지할 수 있습니다.
- **실시간 동기화**: 설정 화면에서 변경한 내용이 즉시 워커에 반영됩니다.
- **채팅 명령어**: 방송 중 채팅으로도 반복을 관리할 수 있습니다.

## 설정 화면 사용법

### 반복 추가하기

1. 우측 상단의 **+ 추가** 버튼을 클릭합니다.
2. 반복 시간(초)을 입력합니다.
3. 반복할 메시지를 입력합니다. (최대 200자)
4. **저장** 버튼을 클릭합니다.

{/* 이미지: 새 반복 추가 화면 (입력 필드가 활성화된 상태) */}
![반복 추가](/bundles/loop-bundle/loop-2.png)

### 반복 수정하기

1. 수정하려는 항목의 **수정** 버튼을 클릭합니다.
2. 시간 또는 메시지를 수정합니다.
3. **저장** 버튼을 클릭합니다.

{/* 이미지: 반복 수정 화면 (편집 모드가 활성화된 카드) */}
![반복 수정](/bundles/loop-bundle/loop-3.png)

### 반복 시작/중지하기

각 항목 우측의 토글 스위치를 클릭하여 반복을 시작하거나 중지할 수 있습니다.

- **사용중**: 반복이 활성화되어 메시지가 주기적으로 전송됩니다.
- **중단**: 반복이 비활성화된 상태입니다.

## 채팅 명령어

방송 중 채팅으로도 반복을 관리할 수 있습니다.

<Callout type="warn">
채팅 명령어는 **관리자** 권한이 있는 사용자만 사용할 수 있습니다.
(매니저 또는 DJ)
</Callout>

### 명령어 목록

| 명령어 | 설명 |
|--------|------|
| `!반복` | 등록된 반복 목록을 확인합니다. |
| `!반복/추가 [시간] [메시지]` | 새로운 반복을 추가합니다. |
| `!반복/삭제 [번호]` | 해당 번호의 반복을 삭제합니다. |
| `!반복/시작 [번호]` | 해당 번호의 반복을 시작합니다. |
| `!반복/중지 [번호]` | 해당 번호의 반복을 중지합니다. |

### 사용 예시

```
!반복/추가 60 안녕하세요! 이 방송은 매일 밤 9시에 진행됩니다.
```

위 명령어는 60초마다 환영 메시지를 반복 전송하는 설정을 추가합니다.

```
!반복
```

등록된 반복 목록을 확인합니다. 다음과 같은 형식으로 표시됩니다:

```
▶ 1. 60: 안녕하세요! 이 방송은 매일 밤 9시에...
■ 2. 120: 팔로우와 좋아요 부탁드립니다!
```

- `▶`: 현재 실행 중인 반복
- `■`: 중지된 반복

```
!반복/시작 2
```

2번 반복을 시작합니다.

```
!반복/중지 1
```

1번 반복을 중지합니다.

```
!반복/삭제 1
```

1번 반복을 삭제합니다.

<Callout type="info">
`!반복/제거`는 `!반복/삭제`와 동일하게 동작합니다.
`!반복/사용`은 `!반복/시작`과 동일하게 동작합니다.
`!반복/중단`은 `!반복/중지`와 동일하게 동작합니다.
</Callout>
