# 룰렛 시스템

룰렛은 청취자들에게 다양한 보상을 제공하는 재미있는 이벤트 시스템입니다. DJ가 설정한 조건에 따라 티켓을 받고, 룰렛을 돌려 아이템을 획득할 수 있습니다.

## 룰렛 티켓 획득 방법

룰렛 티켓은 DJ가 설정한 템플릿의 조건을 만족하면 자동으로 지급됩니다.

### 1. 좋아요 방식
방송에 좋아요를 누르면 티켓을 획득합니다.
- **조건**: 좋아요 1회
- **티켓**: 1장

### 2. 스티커 방식
특정 스티커를 선물하면 티켓을 획득합니다.
- **조건**: DJ가 지정한 스티커 선물
- **분배 모드**: 
  - 활성화: 선물한 사람 모두에게 티켓 지급
  - 비활성화: 선물한 사람에게만 티켓 지급

### 3. 스푼 방식
설정된 개수 이상의 스푼을 선물하면 티켓을 획득합니다.
- **조건**: DJ가 지정한 스푼 개수 이상
- **분배 모드**: 스티커 방식과 동일

![roulette_1](/bundles/starter-pack/roulette_1.png)

## 청취자 명령어

### !룰렛
현재 활성화된 룰렛 템플릿 목록을 확인합니다.

```
!룰렛
```

표시 내용:
```
[룰렛 목록]
1. 스티커 룰렛 (스티커 폭죽)
2. 스푼 룰렛 (스푼 100개 분배)
3. 좋아요 룰렛 (좋아요)
```

![roulette_2](/bundles/starter-pack/roulette_2.png)

### !룰렛 [템플릿 번호] [횟수]
특정 룰렛을 지정한 횟수만큼 돌립니다.

```
!룰렛 1 5
```

- **템플릿 번호**: 룰렛 목록에서 확인한 번호
- **횟수**: 사용할 티켓 개수
- **제한**: 보유한 티켓 개수 이내

### !룰렛 [템플릿 번호]
특정 룰렛의 모든 티켓을 사용합니다.

```
!룰렛 2
```

- **자동 계산**: 보유한 모든 티켓 사용

### !룰렛 전체
모든 룰렛의 티켓을 한 번에 사용합니다.

```
!룰렛 전체
```

- **모든 템플릿**: 각 템플릿별 보유 티켓 모두 사용
- **통합 결과**: 전체 당첨 결과를 한 번에 표시

![roulette_3](/bundles/starter-pack/roulette_3.png)

## 킵(보관) 시스템

룰렛에서 당첨된 커스텀 아이템은 자동으로 보관됩니다. (실드와 복권은 즉시 지급)

### !킵
보관 중인 당첨 아이템 목록을 확인합니다.

```
!킵
```

표시 내용:
```
[보관함]
1. 음료수 쏘기
2. 노래 신청권
3. 별풍선 100개
```

![roulette_4](/bundles/starter-pack/roulette_4.png)

### !사용 [번호]
보관 중인 아이템을 사용합니다.

```
!사용 1
```

- **번호**: 킵 목록에서 확인한 번호
- **처리**: 해당 아이템이 목록에서 제거되고 "사용됨" 상태로 변경
- **DJ 확인**: DJ가 관리 페이지에서 사용 내역 확인 가능

![roulette_5](/bundles/starter-pack/roulette_5.png)

## 특수 아이템

### 실드 아이템
룰렛에서 실드 아이템 당첨 시 즉시 실드 개수가 증감됩니다.

```
축하합니다! 실드 +5 당첨! (현재 실드: 10개)
```

- **즉시 적용**: 킵에 저장되지 않고 바로 적용
- **증감 가능**: +10, -5 등 다양한 설정 가능
- **음수 허용**: 실드 개수는 음수도 가능

### 복권 아이템
룰렛에서 복권 아이템 당첨 시 즉시 복권이 지급됩니다.

```
축하합니다! 복권 3장 당첨!
```

- **즉시 지급**: 킵에 저장되지 않고 바로 지급
- **개수 설정**: DJ가 개수 설정 가능

## 룰렛 템플릿 설정

웹 관리 페이지에서 룰렛 템플릿을 생성하고 관리할 수 있습니다.

### 템플릿 생성 방법
1. 메인 메뉴 > "템플릿 설정" 클릭
2. "새 템플릿" 버튼 클릭
3. 설정 입력:
   - 템플릿 이름
   - 룰렛 방식 (스티커/스푼/좋아요)
   - 분배 모드
   - 자동 실행 여부
   - 룰렛 활성화 여부
4. 아이템 추가:
   - 아이템 타입 선택 (실드/복권/커스텀)
   - 이름 또는 증감값 입력
   - 확률 설정 (%)
5. "저장" 버튼 클릭

![roulette_6](/bundles/starter-pack/roulette_6.png)

### 설정 항목 상세

#### 템플릿 이름
룰렛의 이름을 설정합니다. (예: 스티커 룰렛, 좋아요 이벤트)

#### 룰렛 방식
- **스티커**: 특정 스티커를 받으면 티켓 지급
- **스푼**: 설정한 개수 이상의 스푼을 받으면 티켓 지급
- **좋아요**: 좋아요를 누르면 티켓 지급 (자동 실행 고정)

#### 분배 모드 (스티커/스푼만)
- **활성화**: 선물한 사람 전원에게 티켓 지급
- **비활성화**: 선물한 사람에게만 티켓 지급

#### 자동 실행
- **활성화**: 티켓 받자마자 자동으로 룰렛 실행
- **비활성화**: 청취자가 명령어로 수동 실행
- **좋아요 모드**: 항상 자동 실행 (변경 불가)

#### 룰렛 활성화
- **활성화**: 청취자가 티켓을 받고 사용 가능
- **비활성화**: 일시적으로 룰렛 중지 (기존 티켓은 유지)

![roulette_7](/bundles/starter-pack/roulette_7.png)

### 아이템 타입

#### 실드 아이템
- **증감값**: 정수 입력 (음수 가능)
- **자동 적용**: 당첨 시 즉시 실드 증감
- **표시**: 실드 +10, 실드 -5 형태

#### 복권 아이템
- **개수**: 지급할 복권 개수
- **자동 지급**: 당첨 시 즉시 복권 지급
- **표시**: 복권 3장 형태

#### 커스텀 아이템
- **라벨**: 아이템 이름 (예: 음료수 쏘기, 노래 신청권)
- **킵 저장**: 당첨 시 보관함에 저장
- **사용**: 청취자가 `!사용` 명령어로 사용

### 확률 설정
- **범위**: 0.001% ~ 100%
- **총합**: 100% 이하여야 함
- **꽝**: 나머지 확률은 자동으로 꽝 처리
- **예시**: 
  - 아이템A: 10%
  - 아이템B: 5%
  - 아이템C: 0.5%
  - 꽝: 84.5% (자동 계산)

### 템플릿 수정/삭제
- **수정**: 템플릿 카드의 수정 버튼 클릭
- **삭제**: 템플릿 카드의 삭제 버튼 클릭 후 확인

## 데이터 마이그레이션

기존에 다른 룰렛 번들을 사용하고 있었다면, 데이터를 이전할 수 있습니다.

### 마이그레이션 옵션

#### 1. 공식 룰렛
공식 룰렛 번들의 설정을 가져옵니다.
- **이전 내용**: 템플릿 설정만 이전
- **기록**: 기록은 이전되지 않음

#### 2. [gloomy] 룰렛 자동킵 버전
gloomy 룰렛 번들의 설정과 킵 기록을 가져옵니다.
- **이전 내용**: 템플릿 설정 + 킵 기록
- **미사용 아이템**: 미사용 상태로 이전

#### 3. 🎆하 늘 𝓂𝄞𝓇 룰렛🎆
하늘미르 룰렛 번들의 설정과 기록을 가져옵니다.
- **이전 내용**: 최대 4개 템플릿 + 킵 기록
- **다중 템플릿**: 여러 룰렛 템플릿 동시 이전

#### 4. [crown] 룰렛과 기록
crown 룰렛 번들의 설정을 가져옵니다.
- **이전 내용**: 템플릿 설정만 이전
- **기록**: 기록은 이전되지 않음

![roulette_8](/bundles/starter-pack/roulette_8.png)

### 마이그레이션 방법
1. 템플릿 설정 페이지로 이동
2. "룰렛 템플릿 마이그레이션" 버튼 클릭
3. 이전할 번들 선택
4. 미리보기에서 데이터 확인
5. "마이그레이션 하기" 버튼 클릭
6. 완료될 때까지 대기

### 주의사항
- 같은 번들을 중복으로 이전할 수 없습니다
- 기존 데이터에 추가되는 방식입니다
- 마이그레이션 중에는 창을 닫지 마세요
- 기록이 많을수록 시간이 오래 걸릴 수 있습니다
