# 복권 시스템

복권은 3자리 숫자를 맞추는 미니게임입니다. 복권을 사용하여 당첨되면 경험치를 획득할 수 있습니다.

## 복권 획득 방법

### 1. 레벨업
레벨업 시 복권 1장이 자동으로 지급됩니다.

### 2. 스푼 선물
설정된 개수 이상의 스푼을 선물하면 복권을 획득합니다.
- **기본 설정**: 50개당 1장
- **계산 방식**: (총 스푼 개수 ÷ 설정 개수) = 복권 개수 (소수점 버림)

예시: 설정이 50개일 때
- 49개 선물 → 0장
- 50개 선물 → 1장
- 125개 선물 → 2장

### 3. 룰렛 당첨
룰렛에서 복권 아이템에 당첨되면 복권을 획득합니다.

### 4. DJ 지급
DJ가 특정 청취자 또는 전체 청취자에게 복권을 지급할 수 있습니다.

## 청취자 명령어

### !복권 [숫자1] [숫자2] [숫자3]
복권을 사용하여 당첨 여부를 확인합니다.

```
!복권 1 2 3
```

- **입력 범위**: 0~9 사이의 숫자 3개
- **소모**: 복권 1장 사용

#### 당첨 기준
컴퓨터가 무작위로 3자리 숫자를 추첨하며, 맞춘 개수에 따라 경험치를 획득합니다.

| 맞춘 개수 | 지급 경험치 |
|----------|-----------|
| 0개 | 0점 |
| 1개 | 10점 |
| 2개 | 100점 |
| 3개 | 1,000점 |

![lottery_1](/bundles/starter-pack/lottery_1.png)


### !복권 자동
보유한 모든 복권을 한 번에 사용합니다.

```
!복권 자동
```

- **자동 번호**: [0, 1, 2]로 고정
- **결과 표시**: 최종 통합 결과만 표시
- **편리한 사용**: 복권이 많을 때 유용

예시 결과:
```
[복권 결과]
총 5장 사용
- 3개 맞춤: 0회
- 2개 맞춤: 1회
- 1개 맞춤: 2회
- 0개 맞춤: 2회

총 획득 경험치: 120점
```

## DJ 전용 명령어

### !복권지급 전체 [개수]
현재 방송에 참여 중인 모든 등록된 청취자에게 복권을 지급합니다.

```
!복권지급 전체 3
```

- **대상**: 현재 방송방에 있는 + 애청지수 등록된 청취자
- **개수**: 1명당 지급할 복권 개수

### !복권지급 [고유닉] [개수]
특정 청취자에게 복권을 지급합니다.

```
!복권지급 raravel 5
```

- **고유닉**: 대상 청취자의 고유 태그
- **개수**: 지급할 복권 개수

## 복권 양도

### !복권양도 [고유닉] [수량]
내 복권을 다른 청취자에게 양도합니다.

```
!복권양도 raravel 2
```

- **제한**: 보유한 복권 개수 이내
- **고유닉**: 받을 사람의 고유 태그
- **수량**: 양도할 복권 개수

## 복권 설정

### 애청지수 설정에서 변경
1. 메인 메뉴 > "애청지수 설정" 클릭
2. "복권 시스템" 섹션에서 설정

### 설정 항목
- **복권 시스템**: 활성화/비활성화
- **복권 지급 스푼 개수**: 스푼 몇 개당 복권 1장 지급 (기본 50개)

### 주의사항
- 복권 시스템을 비활성화해도 기존 복권은 사용 가능합니다
- 설정 변경은 즉시 적용됩니다
