# 명령어 레퍼런스

모든 채팅 명령어를 한눈에 확인할 수 있는 레퍼런스 페이지입니다.

## 애청지수 명령어

### 모든 청취자

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `!내정보 생성` | 애청지수 시스템에 등록 | `!내정보 생성` |
| `!내정보 삭제` | 애청지수 시스템에서 탈퇴 | `!내정보 삭제` |
| `!내정보` | 내 애청지수 정보 확인 | `!내정보` |
| `!랭크` | 상위 5명 랭킹 및 채팅왕/좋아요왕 확인 | `!랭크` |

### DJ 전용

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `!상점 [고유닉] [점수]` | 특정 청취자에게 점수 부여 | `!상점 raravel 1000` |
| `!감점 [고유닉] [점수]` | 특정 청취자의 점수 차감 | `!감점 raravel 500` |

## 복권 명령어

### 모든 청취자

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `!복권 [숫자1] [숫자2] [숫자3]` | 복권 사용 (0~9 사이 숫자 3개) | `!복권 1 2 3` |
| `!복권 자동` | 보유한 모든 복권 자동 사용 | `!복권 자동` |
| `!복권양도 [고유닉] [수량]` | 복권을 다른 청취자에게 양도 | `!복권양도 raravel 2` |

### DJ 전용

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `!복권지급 전체 [개수]` | 방에 있는 모든 등록 청취자에게 복권 지급 | `!복권지급 전체 3` |
| `!복권지급 [고유닉] [개수]` | 특정 청취자에게 복권 지급 | `!복권지급 raravel 5` |

## 룰렛 명령어

### 모든 청취자

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `!룰렛` | 활성화된 룰렛 템플릿 목록 확인 | `!룰렛` |
| `!룰렛 [템플릿번호] [횟수]` | 특정 룰렛을 지정 횟수만큼 실행 | `!룰렛 1 5` |
| `!룰렛 [템플릿번호]` | 특정 룰렛의 모든 티켓 사용 | `!룰렛 2` |
| `!룰렛 전체`, `!룰렛 자동` | 모든 룰렛의 모든 티켓 사용 | `!룰렛 전체`, `!룰렛 자동` |
| `!킵` | 보관 중인 당첨 아이템 목록 확인 | `!킵` |
| `!사용 [번호]` | 보관 중인 아이템 사용 | `!사용 1` |

## 실드 명령어

### 모든 청취자

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `!실드` | 현재 실드 개수 확인 | `!실드` |

### DJ/매니저 전용

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `!실드 + [숫자]` | 실드 개수 증가 | `!실드 + 5` |
| `!실드 - [숫자]` | 실드 개수 감소 | `!실드 - 3` |

## 명령어 형식 주의사항

### 띄어쓰기
모든 명령어는 띄어쓰기가 중요합니다.

✅ **올바른 형식**
```
!실드 + 10
!복권 1 2 3
!룰렛 1 5
```

❌ **잘못된 형식**
```
!실드+10
!복권123
!룰렛1 5
```

### 숫자 형식
- **정수만 가능**: 소수점 불가
- **음수 가능**: 실드 증감에서만 (예: `!실드 - 10`)
- **범위 제한**: 복권은 0~9 사이 숫자만

### 고유닉
- **대소문자 구분**: 정확한 고유닉 입력 필요
- **확인 방법**: 스푼라디오에서 `!고유닉` 명령어로 확인
- **공백 불가**: 고유닉에는 공백이 없음
