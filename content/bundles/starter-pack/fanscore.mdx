애청지수는 청취자들의 방송 참여도를 점수화하여 레벨로 표현하는 시스템입니다. 채팅, 좋아요, 스푼 선물 등의 활동으로 경험치를 획득하고 레벨업할 수 있습니다.

## 청취자 명령어

### 내 정보 관리

#### !내정보 생성
애청지수 시스템에 등록합니다. **이 명령어를 입력해야 점수가 누적됩니다.**

```
!내정보 생성
```

#### !내정보 삭제
애청지수 시스템에서 탈퇴합니다. 모든 데이터가 삭제됩니다.

```
!내정보 삭제
```

#### !내정보
현재 내 애청지수 정보를 확인합니다.

```
!내정보
```

표시 정보:
- 레벨
- 현재 경험치
- 다음 레벨까지 필요한 경험치
- 채팅 횟수
- 좋아요 횟수
- 스푼 선물 개수
- 룰렛 개수
- 보유 복권 개수

![fanscore_1](/bundles/starter-pack/fanscore_1.png)

### 랭킹 확인

#### !랭크
상위 5명의 랭킹과 채팅왕, 좋아요왕을 확인합니다.

```
!랭크
```

표시 정보:
- 상위 5명의 닉네임, 레벨, 경험치
- 채팅왕: 가장 많은 채팅을 한 청취자
- 좋아요왕: 가장 많은 좋아요를 누른 청취자

![fanscore_2](/bundles/starter-pack/fanscore_2.png)

## 점수 획득 방법

### 1. 출석 점수
방송에 처음 채팅을 입력하면 출석 점수를 획득합니다.
- **1회 한정**: 방송당 1회만 지급
- **기본값**: 10점

### 2. 채팅 점수
채팅을 입력할 때마다 점수를 획득합니다.
- **기본값**: 1점/채팅

### 3. 좋아요 점수
방송에 좋아요를 누르면 점수를 획득합니다.
- **기본값**: 10점/좋아요

### 4. 스푼 점수
스푼을 선물하면 개수에 비례한 점수를 획득합니다.
- **기본값**: 100점/스푼

### 5. 퀴즈 보너스
DJ가 설정한 간격으로 돌발 퀴즈가 출제됩니다. 가장 먼저 정답을 맞춘 사람에게 보너스 점수가 지급됩니다.
- **자동 출제**: 설정된 시간 간격으로 자동 출제
- **제한 시간**: 기본 5초
- **보너스 점수**: DJ가 설정 (기본값 있음)

## 레벨 시스템

### 레벨업 조건
경험치를 쌓아서 레벨업할 수 있습니다. 레벨이 올라갈수록 더 많은 경험치가 필요합니다.

- **레벨업 공식**: 레벨² × 100 경험치 필요
- 예시:
  - Lv.1 → Lv.2: 100 경험치
  - Lv.2 → Lv.3: 400 경험치
  - Lv.3 → Lv.4: 900 경험치

### 레벨업 보상
레벨업 시 복권 1장이 자동으로 지급됩니다.

```
{닉네임}님. 레벨업 하셨습니다! 현재 Lv. {레벨}
```

## DJ 전용 명령어

### 점수 부여/차감

#### !상점 [고유닉] [점수]
특정 청취자에게 점수를 부여합니다.

```
!상점 raravel 1000
```

#### !감점 [고유닉] [점수]
특정 청취자의 점수를 차감합니다.

```
!감점 raravel 500
```

> **고유닉이란?** 사용자의 고유 태그입니다. `!고유닉` 명령어로 확인할 수 있습니다.

## 애청지수 설정 페이지

웹 관리 페이지에서 애청지수 시스템을 상세하게 설정할 수 있습니다.

### 접근 방법
1. 메인 메뉴에서 "애청지수 설정" 클릭
2. 각 항목별 설정 조정

### 설정 항목

#### 애청지수 시스템
- **활성화/비활성화**: 전체 시스템 켜기/끄기

#### 점수 설정
- **출석 점수**: 방송 첫 채팅 시 지급 점수
- **채팅 1회당 점수**: 채팅당 지급 점수
- **좋아요 1회당 점수**: 좋아요당 지급 점수
- **스푼 1개당 점수**: 스푼 선물당 지급 점수

#### 돌발 퀴즈
- **활성화/비활성화**: 퀴즈 시스템 켜기/끄기
- **퀴즈 실행 간격**: 퀴즈 출제 간격 (최소 30초, 기본 180초)
- **퀴즈 정답 입력 시간**: 정답 입력 제한 시간 (기본 5초)
- **보너스 점수**: 정답자에게 지급할 점수
- **퀴즈 목록**: 질문과 답변 추가/삭제

#### 복권 시스템
- **활성화/비활성화**: 복권 시스템 켜기/끄기
- **복권 지급 스푼 개수**: 스푼 선물로 복권 획득 시 필요한 스푼 개수 (기본 50개)

![fanscore_4](/bundles/starter-pack/fanscore_4.png)

### 퀴즈 관리

#### 퀴즈 추가
1. "퀴즈 추가" 버튼 클릭
2. 질문 입력
3. 정답 입력
4. "저장" 버튼 클릭

#### 퀴즈 삭제
각 퀴즈 항목의 삭제 버튼(X)을 클릭하여 삭제할 수 있습니다.

![fanscore_5](/bundles/starter-pack/fanscore_5.png)

## 데이터 마이그레이션

기존에 다른 애청지수 번들을 사용하고 있었다면, 데이터를 이전할 수 있습니다.

### 마이그레이션 방법
1. 애청지수 설정 페이지로 이동
2. "애청지수 데이터 불러오기" 버튼 클릭
3. 기존 번들 데이터 확인
4. "마이그레이션 하기" 버튼 클릭
5. 완료될 때까지 대기

### 주의사항
- 기존 데이터가 있는 경우 덮어쓰기됩니다
- 마이그레이션 중에는 창을 닫지 마세요
- 청취자가 많을수록 시간이 오래 걸릴 수 있습니다
