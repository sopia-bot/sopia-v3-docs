# 관리 페이지

웹 관리 페이지에서 모든 기능을 설정하고 데이터를 확인할 수 있습니다. 방송 중이 아니어도 언제든지 접속하여 관리할 수 있습니다.

## 대시보드

메인 화면에서 방송의 주요 통계를 한눈에 확인할 수 있습니다.

### 표시 정보

#### 주요 통계 카드
- **총 등록 청취자**: 애청지수 시스템에 등록된 청취자 수
- **오늘 활동한 청취자 수**: 오늘 방송에 참여한 청취자 수
- **총 복권 보유**: 모든 청취자가 보유한 복권 총합
- **현재 실드**: 현재 실드 개수

![management_1](/bundles/starter-pack/management_1.png)

#### 애청지수 랭킹 (Top 5)
상위 5명의 청취자를 보여줍니다.
- 순위
- 닉네임
- 레벨
- 경험치

![management_2](/bundles/starter-pack/management_2.png)

#### 최근 룰렛 현황
최근 5개의 룰렛 당첨 내역을 보여줍니다.
- 닉네임
- 템플릿 이름
- 당첨 아이템
- 확률
- 시간
- 사용 상태

![management_3](/bundles/starter-pack/management_3.png)

### 새로고침
우측 상단의 새로고침 버튼을 클릭하여 최신 데이터로 업데이트할 수 있습니다.

## 청취자 관리

등록된 모든 청취자의 정보를 확인하고 관리할 수 있습니다.

### 청취자 목록
다음 정보를 표로 확인할 수 있습니다:
- **닉네임**: 청취자 닉네임
- **고유닉**: 고유 태그
- **레벨**: 현재 레벨
- **애청지수**: 현재 경험치
- **채팅**: 총 채팅 횟수
- **좋아요**: 총 좋아요 횟수
- **스푼**: 총 선물한 스푼 개수
- **복권**: 보유한 복권 개수
- **룰렛**: 보유한 룰렛 티켓 개수 (템플릿별)
- **마지막 활동 시간**: 마지막 활동 일시

![management_4](/bundles/starter-pack/management_4.png)

### 필터링 기능

#### 사용자 필터
특정 청취자만 표시할 수 있습니다.
- **선택 방법**: 드롭다운에서 닉네임 선택
- **다중 선택**: 여러 명 선택 가능
- **전체 선택**: 기본값

#### 레벨 필터
특정 레벨 조건의 청취자만 표시할 수 있습니다.
- **이상**: 선택한 레벨 이상
- **이하**: 선택한 레벨 이하
- **예시**: "Lv.5 이상" 선택 시 레벨 5 이상만 표시

#### 마지막 활동 시간 필터
특정 기간에 활동한 청취자만 표시할 수 있습니다.
- **시작 시간**: 필터 시작 일시
- **종료 시간**: 필터 종료 일시 (비워두면 현재 시각)
- **예시**: "2024-01-01 ~ 2024-01-31" 선택 시 1월 활동자만 표시

![management_5](/bundles/starter-pack/management_5.png)

### 정렬 기능
열 제목을 클릭하여 정렬할 수 있습니다.
- **정렬 가능 열**: 레벨, 애청지수, 채팅, 좋아요, 스푼
- **오름차순/내림차순**: 화살표 아이콘으로 표시
- **클릭**: 오름차순 → 내림차순 → 정렬 해제

### 페이지네이션
많은 청취자가 있을 때 페이지 단위로 나누어 표시됩니다.
- **페이지 크기**: 10명 단위
- **페이지 이동**: 하단 버튼으로 이동

### 검색 기능
닉네임이나 고유닉으로 청취자를 검색할 수 있습니다.
- **검색창**: 상단의 검색 입력란
- **실시간 검색**: 입력하는 즉시 필터링

## 룰렛 기록

모든 룰렛 당첨 내역을 확인하고 관리할 수 있습니다.

### 기록 목록
다음 정보를 표로 확인할 수 있습니다:
- **닉네임**: 당첨자 닉네임
- **템플릿**: 룰렛 템플릿 이름
- **당첨 아이템**: 당첨된 아이템 이름
- **확률**: 당첨 확률
- **상태**: 사용됨/미사용
- **시간**: 당첨 일시

![management_6](/bundles/starter-pack/management_6.png)

### 필터링 기능

#### 상태 필터
사용 상태별로 필터링할 수 있습니다.
- **전체**: 모든 기록 표시
- **미사용**: 아직 사용하지 않은 아이템
- **사용됨**: 이미 사용한 아이템

#### 템플릿 필터
특정 룰렛 템플릿의 기록만 표시할 수 있습니다.
- **선택 방법**: 드롭다운에서 템플릿 선택
- **전체 선택**: 기본값

#### 사용자 필터
특정 청취자의 당첨 기록만 표시할 수 있습니다.
- **선택 방법**: 드롭다운에서 닉네임 선택
- **검색 기능**: 닉네임으로 검색 가능

![management_7](/bundles/starter-pack/management_7.png)

### 정렬 기능
열 제목을 클릭하여 정렬할 수 있습니다.
- **정렬 가능 열**: 당첨 아이템, 확률, 시간
- **기본 정렬**: 시간 내림차순 (최신순)

### 상태 변경
DJ가 청취자의 아이템 사용 상태를 변경할 수 있습니다.
- **토글 버튼**: 상태 열의 스위치 클릭
- **용도**: 아이템을 사용했을 때 "사용됨"으로 변경

### 꽝 기록
"꽝"은 기록에 남지 않습니다. 당첨된 아이템만 기록됩니다.
