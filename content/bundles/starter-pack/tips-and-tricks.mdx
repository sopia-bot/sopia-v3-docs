# 활용 팁

애청지수 도구를 더욱 효과적으로 활용하기 위한 팁과 노하우를 소개합니다.

## DJ 방송 운영 팁

### 애청지수 시스템 활용

#### 1. 적절한 점수 밸런스
청취자들이 너무 빨리 레벨업하거나 너무 느리게 성장하지 않도록 점수를 조절하세요.

**추천 설정 (일반 방송)**
- 출석 점수: 10점
- 채팅 점수: 1점
- 좋아요 점수: 10점
- 스푼 점수: 100점
- 퀴즈 보너스: 50점

**추천 설정 (소규모 방송)**
- 출석 점수: 20점
- 채팅 점수: 2점
- 좋아요 점수: 20점
- 스푼 점수: 200점
- 퀴즈 보너스: 100점

#### 2. 퀴즈 활용 전략
- **간격 조절**: 너무 자주 출제하면 성가실 수 있으니 3~5분 간격 권장
- **난이도 조절**: 쉬운 퀴즈와 어려운 퀴즈를 섞어서 등록
- **방송 관련 퀴즈**: 방송 컨셉이나 DJ 정보로 재미 요소 추가

#### 3. 레벨업 이벤트
특정 레벨에 도달한 청취자에게 특별 보상 제공:
- Lv.10 달성 시 특별 멘트
- Lv.20 달성 시 노래 신청권
- Lv.50 달성 시 닉네임 변경권

### 룰렛 시스템 활용

#### 1. 다양한 템플릿 운영
목적에 맞게 여러 룰렛을 동시에 운영하세요.

**예시 구성**
- 좋아요 룰렛: 가벼운 보상 (자동 실행)
- 스티커 룰렛: 중간 보상 (수동 실행)
- 고액 스푼 룰렛: 큰 보상 (수동 실행)

#### 2. 확률 설정 노하우
- **꽝 비율**: 40~60% 권장 (너무 높으면 재미 감소)
- **고가 보상**: 0.1~1% (레어템 느낌)
- **중가 보상**: 5~10% (적당한 당첨 빈도)
- **저가 보상**: 20~30% (자주 당첨되는 기본 보상)

#### 3. 아이템 구성 팁
다양한 타입의 아이템을 섞어서 재미를 높이세요:
- 실드 증감: ±5, ±10
- 복권 지급: 1~5장
- 커스텀 아이템: 음료수, 노래 신청, 멘트, 이모티콘 등

#### 4. 시즌별 이벤트
특별한 날에는 특별 템플릿 추가:
- 생일: 특별 보상 확률 높이기
- 기념일: 한정 아이템 추가
- 이벤트: 일시적으로 당첨 확률 상향

### 실드 시스템 활용

#### 1. 미니게임
실드를 활용한 간단한 게임 진행:
- **업다운 게임**: 정답 맞추면 +1, 틀리면 -1
- **릴레이 채팅**: 특정 단어 금지, 사용 시 -5
- **응원 게임**: 좋아요 100개당 +10

#### 2. 목표 달성 이벤트
청취자들과 함께 실드 목표 달성:
```
목표: 실드 100개 달성
현재: 45개
달성 시: 노래 3곡 추가
```

#### 3. 벌칙 시스템
재미있는 벌칙으로 긴장감 추가:
```
실드 0개 이하 시: DJ 벌칙
실드 -10개 이하 시: 방송 연장 10분
```

### 복권 시스템 활용

#### 1. 복권 대회
특정 시간에 복권 대회 개최:
```
오늘 밤 10시 복권 대회!
참가 방법: !복권 자동
최다 당첨자에게 선물!
```

#### 2. 복권 나눔 이벤트
DJ가 특정 조건으로 복권 지급:
```
!복권지급 전체 5
(모든 청취자에게 5장씩 지급)
```

## 데이터 관리 팁

### DJ를 위한 관리

#### 1. 정기적인 확인
- 주 1회 청취자 관리 페이지 확인
- 활동하지 않는 청취자 파악
- 상위 랭커 변동 확인

#### 2. 필터 활용
- 레벨 필터로 초보/중수/고수 구분
- 활동 시간 필터로 최근 방문자 확인
- 복권/티켓 보유량으로 이벤트 대상 선정

#### 3. 기록 관리
- 룰렛 기록으로 사용하지 않은 아이템 확인
- 실드 변경 기록으로 이벤트 진행 파악
- 통계 자료로 방송 분석

### 백업 및 마이그레이션

#### 1. 정기 백업
관리 페이지를 스크린샷으로 저장하여 백업:
- 청취자 목록
- 템플릿 설정
- 주요 통계

#### 2. 마이그레이션 활용
- 다른 번들에서 데이터 이전
- 미리보기로 데이터 확인 후 진행
- 진행 중 창 닫지 않기

## 커뮤니티 형성 팁

### 경쟁과 협동

#### 1. 랭킹 경쟁
- 주간/월간 랭킹 1위에게 보상
- 채팅왕/좋아요왕에게 특별 혜택
- 레벨 달성자 축하

#### 2. 팀 이벤트
청취자를 팀으로 나누어 경쟁:
- 팀별 애청지수 합산
- 팀별 실드 목표 달성
- 팀별 복권 대회

### 신규 유입 유도

#### 1. 초보자 혜택
- 신규 등록 시 복권 지급
- Lv.1~5 빠른 성장 지원
- 초보자 가이드 제공

#### 2. 추천인 제도
추천받아 가입한 청취자와 추천인에게 보상:
- 추천인: 복권 5장
- 신규 유입: 복권 3장

## 문제 상황 대처

### 점수 오류 발생 시
1. 해당 청취자의 점수 확인
2. `!상점` 또는 `!감점`으로 수동 조정
3. 조정 사유 청취자에게 설명

### 룰렛 오작동 시
1. 템플릿 설정 확인
2. 확률 합계 확인 (100% 이하)
3. 청취자에게 보상 수동 지급

### 시스템 과부하 시
1. 퀴즈 간격 늘리기
2. 배치 처리 대기 (최대 5초)
3. 불필요한 템플릿 비활성화
