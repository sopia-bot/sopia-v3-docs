# 실드 시스템

실드는 DJ가 방송 중 관리할 수 있는 숫자 카운터입니다. 채팅 명령어로 실드 개수를 증가시키거나 감소시킬 수 있으며, 룰렛과 연동하여 다양하게 활용할 수 있습니다.

## 모든 청취자 명령어

### !실드
현재 실드 개수를 확인합니다.

```
!실드
```

표시 내용:
```
현재 실드: 10개
```

![shield_1](/bundles/starter-pack/shield_1.png)

## DJ/매니저 전용 명령어

방송 매니저만 실드를 증감할 수 있습니다.

### !실드 + [숫자]
실드 개수를 증가시킵니다.

```
!실드 + 5
```

표시 내용:
```
실드가 5개 증가했습니다! (현재 실드: 15개)
```

### !실드 - [숫자]
실드 개수를 감소시킵니다.

```
!실드 - 3
```

표시 내용:
```
실드가 3개 감소했습니다! (현재 실드: 12개)
```

### 음수 허용
실드 개수는 음수도 가능합니다.

```
!실드 - 20
```

결과:
```
실드가 20개 감소했습니다! (현재 실드: -8개)
```

## 룰렛과의 연동

실드를 룰렛 아이템으로 설정하여 청취자들이 룰렛으로 실드를 얻거나 잃게 할 수 있습니다.

### 룰렛 아이템 설정 예시

#### 실드 증가 아이템
- **타입**: 실드
- **증감값**: +5
- **확률**: 10%
- **효과**: 당첨 시 실드 5개 증가

#### 실드 감소 아이템
- **타입**: 실드
- **증감값**: -10
- **확률**: 5%
- **효과**: 당첨 시 실드 10개 감소

![shield_3](/bundles/starter-pack/shield_3.png)

### 당첨 시 처리
룰렛에서 실드 아이템 당첨 시:
1. 실드 개수 즉시 증감
2. 채팅으로 결과 알림
3. 킵에 저장되지 않음 (즉시 적용)

```
축하합니다! 실드 +5 당첨! (현재 실드: 20개)
```

## 실드 설정 페이지

웹 관리 페이지에서 실드 시스템을 관리할 수 있습니다.

### 접근 방법
1. 메인 메뉴에서 "실드 설정" 클릭
2. 실드 개수 및 설정 확인

### 설정 항목

#### 실드 시스템
- **활성화/비활성화**: 실드 시스템 켜기/끄기

#### 현재 실드 개수
- **수동 설정**: 특정 값으로 직접 설정 가능
- **초기화**: 0으로 리셋

#### 실드 변경 기록
최근 실드 변경 내역을 확인할 수 있습니다.
- **시간**: 변경된 시각
- **변경 내용**: 증감 수량
- **변경 후 값**: 변경 후 실드 개수
- **변경자**: 명령어를 입력한 사람

## 주의사항

### 실드 리셋
- 방송을 종료해도 실드 개수는 유지됩니다
- 새로운 이벤트 시작 시 수동으로 리셋 필요
- 웹 페이지에서 "초기화" 버튼으로 간편하게 리셋

### 음수 처리
- 실드가 0 이하로 떨어져도 오류 없이 작동
- 음수 상태에서도 계속 증감 가능
- 이벤트 실패 조건으로 활용 가능

### 명령어 형식
- 띄어쓰기 필수: `!실드 + 10` (O) / `!실드+10` (X)
- 숫자만 입력: 소수점 불가, 정수만 가능
- 기호 필수: `!실드 10` (X) / `!실드 + 10` (O)
