# 소피아 신청곡 번들

## 주요 기능

### 무료 신청곡 관리
- 시청자들이 자유롭게 신청할 수 있는 무료 신청곡 기능
- 사용자당 최대 신청곡 개수 제한
- 신청 간격 시간 제한
- 방송별 제한 초기화 기능

### 유료 신청곡 관리
- 특정 스티커 또는 일정 금액 이상 후원으로 신청곡 허용
- 스티커 선택 기능 (시각적 인터페이스)
- 최소 후원 금액 설정
- 후원 금액에 따른 신청권 배분 기능

### 신청곡 재생 관리
- 현재 재생 중인 곡 표시 (앨범 커버, 아티스트, 제목, 신청자)
- 재생 컨트롤 (재생/이전/다음/정지)
- 셔플 모드
- 진행률 표시

### 유료 신청권 수동 지급
- 현재 방송 참여자 검색 및 선택
- 선택한 사용자에게 유료 신청권 지급
- 티켓 사용 현황 추적
