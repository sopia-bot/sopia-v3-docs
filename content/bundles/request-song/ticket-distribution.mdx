# 유료 신청권 지급

유료 신청권 지급 기능을 사용하면 후원 없이도 특정 시청자에게 유료 신청권을 수동으로 지급할 수 있습니다. 이벤트, 경품, 또는 특별한 경우에 유용하게 사용할 수 있습니다.

## 개요

### 언제 사용하나요?

- **이벤트 경품**: 퀴즈나 게임에서 우승한 시청자에게 보상
- **감사 표시**: 열성 팬이나 기여자에게 감사의 의미로 지급
- **보상**: 방송 참여도가 높은 시청자에게 보상
- **테스트**: 신청곡 시스템을 테스트할 때
- **오류 복구**: 시스템 오류로 신청권을 받지 못한 시청자에게 재지급

### 자동 지급과의 차이

| 구분 | 자동 지급 | 수동 지급 |
|------|----------|----------|
| **트리거** | 후원 (스티커 또는 금액) | 방송자의 수동 선택 |
| **대상** | 후원한 모든 시청자 | 방송자가 선택한 시청자 |
| **개수** | 1개 (또는 배분 모드에 따라) | 1개 (기본) |
| **사용** | 후원과 동시에 지급 | 방송자가 원하는 시점에 지급 |

## 사용 방법

### 1단계: 유료 신청권 지급 화면 열기

1. 설정 패널 또는 메인 화면에서 **"유료 신청권 지급"** 버튼을 찾습니다
2. 버튼을 클릭하여 "방송 참여자 목록" 대화상자를 엽니다

### 2단계: 참여자 목록 확인

대화상자가 열리면 현재 방송에 참여 중인 시청자 목록이 표시됩니다.

#### 표시 정보

- **닉네임**: 시청자의 표시 이름
- **사용자 태그**: 시청자의 고유 ID
- **선택 체크박스**: 신청권을 지급할 시청자 선택

#### 참여자 수 표시

- 대화상자 상단에 **"총 N명"** 형태로 현재 참여자 수가 표시됩니다
- 예: "총 25명"

### 3단계: 참여자 검색

많은 참여자가 있을 때, 검색 기능을 사용하여 원하는 시청자를 빠르게 찾을 수 있습니다.

#### 검색 방법

1. **검색창**에 검색어를 입력합니다
2. 검색어는 다음을 기준으로 필터링됩니다:
   - 닉네임
   - 사용자 태그

#### 검색 예시

**참여자 목록**:
- 닉네임: "홍길동", 태그: "user123"
- 닉네임: "김철수", 태그: "user456"
- 닉네임: "이영희", 태그: "user789"

**검색어**: "길동"
- 결과: "홍길동" (user123)

**검색어**: "user4"
- 결과: "김철수" (user456)

### 4단계: 시청자 선택

신청권을 지급할 시청자를 선택합니다.

#### 단일 선택
1. 원하는 시청자의 **체크박스**를 클릭합니다
2. 체크 표시가 나타나면 선택된 것입니다

#### 다중 선택
1. 여러 시청자의 **체크박스**를 클릭하여 다중 선택할 수 있습니다
2. 선택된 시청자는 모두 체크 표시가 나타납니다

#### 선택 취소
- 체크박스를 다시 클릭하면 선택이 취소됩니다

### 5단계: 참여자 목록 새로고침 (선택 사항)

참여자 목록이 최신 상태가 아니라고 생각되면:

1. **"새로고침"** 버튼을 클릭합니다
2. 목록이 업데이트됩니다
3. 새로 참여한 시청자가 추가되고, 퇴장한 시청자가 제거됩니다

> **참고**: 새로고침하면 선택 상태가 초기화될 수 있습니다.

### 6단계: 신청권 지급

1. 지급할 시청자를 모두 선택한 후, **"확인"** 또는 **"지급"** 버튼을 클릭합니다
2. 선택된 시청자에게 유료 신청권이 지급됩니다
3. 대화상자가 닫힙니다

### 7단계: 취소

지급을 취소하려면:
1. **"취소"** 버튼을 클릭합니다
2. 아무런 변경 없이 대화상자가 닫힙니다
