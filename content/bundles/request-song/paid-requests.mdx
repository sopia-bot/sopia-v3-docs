# 유료 신청곡

유료 신청곡 기능을 사용하면 시청자들이 후원을 통해 신청곡을 요청할 수 있습니다. 특정 스티커 또는 일정 금액 이상의 후원을 받을 때 자동으로 신청곡을 허용할 수 있습니다.

## 기본 설정

### 유료 신청곡 활성화

1. 설정 패널에서 **"유료 신청곡 허용"** 스위치를 찾습니다
2. 스위치를 **켜기(ON)** 상태로 변경합니다
3. 후원 방식을 선택합니다 (특정 스티커 또는 최소 금액)

### 유료 신청곡 비활성화

1. **"유료 신청곡 허용"** 스위치를 **끄기(OFF)** 상태로 변경합니다
2. 시청자들은 더 이상 후원으로 신청곡을 요청할 수 없습니다

## 후원 방식 선택

유료 신청곡을 활성화한 경우, 두 가지 후원 방식 중 하나를 선택할 수 있습니다.

### 방식 1: 특정 스티커

특정 스티커를 선물받으면 자동으로 신청곡을 허용하는 방식입니다.

#### 설정 방법

1. **"후원 방식"**에서 **"특정 스티커"**를 선택합니다
2. **"스티커 선택"** 버튼을 클릭합니다
3. 스티커 선택 대화상자가 열립니다
4. 원하는 스티커를 클릭하여 선택합니다
5. **"확인"** 버튼을 클릭합니다

### 방식 2: 최소 금액

일정 금액 이상의 후원을 받으면 자동으로 신청곡을 허용하는 방식입니다.

#### 설정 방법

1. **"후원 방식"**에서 **"최소 금액"**을 선택합니다
2. **"최소 후원 금액"** 입력란에 금액을 입력합니다
   - 예: `5000`을 입력하면 5,000원 이상 후원 시 신청권 획득
3. 필요에 따라 **"신청권 배분 모드"**를 활성화합니다

## 신청권 배분 모드

**"최소 금액"** 방식에서만 사용할 수 있는 고급 기능입니다.

### 배분 모드란?

후원 금액에 비례하여 여러 개의 신청권을 지급하는 기능입니다.

### 동작 방식

신청권 개수는 다음 공식으로 계산됩니다:

```
신청권 개수 = 후원 금액 ÷ 최소 후원 금액 (내림)
```

## 신청권 관리

### 신청권 확인

신청권은 자동으로 추적되며, 다음 정보가 저장됩니다:

- **사용자**: 신청권을 받은 시청자
- **후원 유형**: 스티커 또는 금액
- **후원 정보**: 스티커 이름 또는 후원 금액
- **사용 여부**: 사용됨 / 미사용

### 신청권 소진

- 시청자가 신청곡을 요청하면 자동으로 신청권이 소진됩니다
- 배분 모드에서는 1곡당 1개의 신청권이 소진됩니다
- 사용된 신청권은 복구할 수 없습니다

### 신청권 수동 지급

후원 없이 특정 시청자에게 신청권을 지급할 수도 있습니다.

자세한 내용은 [유료 신청권 지급](./ticket-distribution.mdx) 페이지를 참고하세요.
