# 명령어 가이드

소피아 신청곡 번들에서 사용할 수 있는 모든 채팅 명령어를 안내합니다.

## 개요

시청자들은 채팅창에 명령어를 입력하여 신청곡을 요청하고 관리할 수 있습니다. 모든 명령어는 한글로 작성되어 있으며, 채팅창에 입력하면 자동으로 처리됩니다.

## 시청자용 명령어

### 1. 신청곡 요청

신청하고 싶은 곡을 요청하는 명령어입니다.

#### 사용 방법

```
신청곡 아티스트 - 곡제목
```

#### 예시

```
신청곡 BTS - Dynamite
신청곡 아이유 - 밤편지
신청곡 NewJeans - Hype Boy
신청곡 임영웅 - 무지개
```

#### 작동 방식

1. 채팅창에 명령어를 입력합니다
2. 시스템이 다음을 자동으로 확인합니다:
   - 무료 신청곡이 허용되어 있는지
   - 곡 개수 제한을 초과하지 않았는지
   - 시간 제한을 준수했는지
   - 유료 신청권이 있는지 (유료 신청의 경우)
3. 조건을 만족하면 신청곡 목록에 추가됩니다
4. 확인 메시지가 표시됩니다

---

### 2. 신청곡 취소

가장 최근에 신청한 곡을 취소하는 명령어입니다.

#### 사용 방법

```
신청곡 취소
```

#### 취소 가능 조건

- ✅ 자신이 신청한 곡이어야 함
- ✅ 아직 재생되지 않은 곡이어야 함
- ✅ 가장 최근에 신청한 곡만 취소 가능

#### 주의사항

- **최근 곡만 취소**: 이전에 신청한 곡은 취소할 수 없습니다
- **재생 전에만 가능**: 이미 재생된 곡은 취소할 수 없습니다
- **환불 자동 처리**: 유료 신청권은 자동으로 환불됩니다

---

### 3. 신청곡 목록 보기

현재 대기 중인 신청곡 목록을 확인하는 명령어입니다.

#### 사용 방법

```
신청곡
```

매개변수 없이 "신청곡"만 입력하면 목록을 볼 수 있습니다.

#### 작동 방식

1. 채팅창에 "신청곡"을 입력합니다
2. 시스템이 현재 대기 중인 신청곡 목록을 가져옵니다
3. 다음 재생 예정인 5곡을 보여줍니다

#### 응답 메시지

```
📋 신청곡 대기 목록:
1. Dynamite - BTS
2. 밤편지 - 아이유
3. Hype Boy - NewJeans
4. 무지개 - 임영웅
5. 사랑은 늘 도망가 - 임영웅
```

목록이 비어있으면:
- "현재 대기 중인 신청곡이 없습니다"

### 4. 현재 재생곡 확인

지금 재생 중인 곡의 정보를 확인하는 명령어입니다.

#### 사용 방법

```
현재곡
```

#### 작동 방식

1. 채팅창에 "현재곡"을 입력합니다
2. 시스템이 현재 재생 중인 곡의 정보를 가져옵니다
3. 곡 정보와 신청자를 표시합니다

#### 응답 메시지

재생 중인 곡이 있을 때:
```
🎵 현재 재생 중:
신청자: 홍길동
곡: Dynamite - BTS
```

재생 중인 곡이 없을 때:
- "현재 재생 중인 곡이 없습니다"
