# 신청곡 관리

신청곡 관리 기능을 사용하여 신청된 곡을 재생하고, 재생 순서를 제어하며, 필요 없는 곡을 삭제할 수 있습니다.

## 화면 구성

신청곡 관리 화면은 두 가지 주요 영역으로 구성됩니다:

1. **현재 재생곡** - 상단에 표시되는 현재 재생 중인 곡 정보
2. **신청곡 목록** - 하단에 표시되는 모든 신청곡 테이블

## 현재 재생곡

### 표시 정보

현재 재생 중인 곡은 다음 정보를 표시합니다:

- **앨범 커버**: 곡의 썸네일 이미지
- **아티스트**: 가수 또는 아티스트 이름
- **곡 제목**: 노래 제목
- **신청자**: 신청한 시청자의 이름
- **진행률**: 현재 재생 시간 / 전체 재생 시간
- **진행 바**: 재생 진행 상태를 시각적으로 표시

### 재생 컨트롤

현재 재생곡 영역에서는 다음 컨트롤을 사용할 수 있습니다:

#### 재생 (▶)
- 현재 선택된 곡을 재생합니다
- 이미 재생 중인 경우, 일시정지 기능으로 변경됩니다

#### 이전 (⏮)
- 이전에 재생한 곡으로 이동합니다
- 재생 기록이 없으면 비활성화됩니다

#### 다음 (⏭)
- 다음 곡으로 이동합니다
- 신청곡 목록에서 아직 재생하지 않은 곡 중 다음 곡을 재생합니다
- 셔플 모드가 켜져 있으면 무작위로 선택합니다

#### 정지 (⏹)
- 현재 재생을 중지합니다
- 재생 상태를 초기화합니다

#### 셔플 (🔀)
- 셔플 모드를 켜거나 끕니다
- **ON 상태**: 다음 곡이 무작위로 선택됩니다
- **OFF 상태**: 신청곡 목록 순서대로 재생됩니다

## 신청곡 목록

### 테이블 구성

신청곡 목록은 테이블 형태로 표시되며, 다음 열로 구성됩니다:

| 열 이름 | 설명 |
|---------|------|
| **재생됨** | 재생 여부 표시 (✓: 재생됨, ✗: 미재생) |
| **추가된 시간** | 신청곡이 추가된 시간 |
| **아티스트** | 가수 또는 아티스트 이름 |
| **곡 제목** | 노래 제목 |
| **신청자** | 신청한 시청자의 이름 |
| **액션** | 재생 및 삭제 버튼 |

### 액션 버튼

각 신청곡 행에는 다음 버튼이 있습니다:

#### 재생 버튼
- **기능**: 해당 곡을 즉시 재생합니다
- **동작**:
  - 현재 재생 중인 곡을 중지하고 선택한 곡을 재생합니다
  - "현재 재생곡" 영역에 해당 곡 정보가 표시됩니다
  - 재생됨 상태가 ✓로 변경됩니다

#### 삭제 버튼
- **기능**: 해당 곡을 신청곡 목록에서 삭제합니다
- **동작**:
  - 목록에서 즉시 삭제됩니다
  - 되돌릴 수 없습니다
  - 현재 재생 중인 곡을 삭제하면 재생이 중지됩니다

### 목록 정렬

신청곡 목록은 기본적으로 **추가된 시간** 순서로 정렬됩니다:
- 최근에 추가된 곡이 상단에 표시됩니다
- 또는 가장 오래된 곡이 상단에 표시됩니다 (설정에 따라 다름)

## 신청곡 삭제

### 개별 삭제

특정 곡만 삭제하려면:

1. 신청곡 목록에서 삭제할 곡을 찾습니다
2. 해당 곡의 **"삭제"** 버튼을 클릭합니다
3. 곡이 즉시 목록에서 제거됩니다

> **주의**: 삭제는 되돌릴 수 없습니다.

### 전체 삭제

모든 신청곡을 한 번에 삭제하려면:

1. **"전체 삭제"** 버튼을 클릭합니다
2. 확인 대화상자가 표시됩니다
3. **"확인"**을 클릭하여 모든 곡을 삭제합니다
4. **"취소"**를 클릭하여 삭제를 중단합니다

> **주의**: 전체 삭제는 되돌릴 수 없으며, 모든 신청곡이 영구적으로 삭제됩니다.


## 음악 검색 및 추가

시청자가 신청곡을 신청할 때, 자동으로 음악 정보를 검색합니다.

### 검색 우선순위

1. **FLO 음악 플랫폼** - 1순위 검색
2. **YouTube** - FLO에서 찾지 못한 경우 2순위 검색

### 검색 정보

다음 정보가 자동으로 추출됩니다:
- 곡 제목
- 아티스트
- 재생 시간
- 썸네일 이미지 (앨범 커버)
