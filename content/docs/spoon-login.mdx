<Callout type="warning" title="방송 계정과 매니저 계정은 별도입니다.">
    방송용 계정으로 소피아에 로그인할 경우
    <br/> 소피아 프로그램에서 방송에 참여할 수 없어 오류가 발생합니다.
</Callout>

## 로그인하기 전에

스푼라디오의 계정과 연동하기 위해선, 웹 브라우저에서 로그인해야 합니다.
<br/> 현재는 [Chrome 브라우저](https://www.google.com/chrome/)만 지원하기 때문에 만약 설치되어있지 않다면 미리 설치해 주세요.

## 로그인하기

![login](/docs/spoon-login.png)

<span className='font-bold text-[#FF4100] underline'>스푼 페이지에서 로그인하기</span>버튼을 누를경우, 크롬에서 스푼페이지로 이동합니다.
<br/>브라우저 내에서 소피아와 연동할 계정으로 로그인 할 경우, 아래와 같은 알림이 나타납니다.

![modal](/docs/spoon-modal.png)


<Accordions>

      아래 이미지와 같은 창이 뜰 경우 허용을 클릭해 주세요.

      ![local-network](/docs/local-network.png)
