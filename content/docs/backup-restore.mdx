# 백업 및 복원

소피아 데이터를 백업하고 복원하는 방법을 알려드립니다.

- 일부 공식 번들이 복원되지 않을 수 있습니다. 
- 백업 기능은 100% 데이터를 보장하지 않습니다. 다만, 안 하는 것보다는 낫습니다. 백업하지 않고 유실된 데이터에 대한 책임은 문의받지 않습니다.

<Callout type="error">
    백업을 생활화 하세요. 데이터 유실 시 복구가 어려울 수 있습니다.
</Callout>

## 백업

<Steps>
<Step>
왼쪽 사이드바에서 <span className='font-bold text-[#1100ff]'>백업 / 복원</span> 메뉴를 클릭합니다.
</Step>
<Step>
새 백업 생성 버튼을 클릭합니다.
</Step>
<Step>
![backup-restore-1](/docs/backup-restore-1.png)

백업할 항목을 선택하고 백업합니다.
</Step>
<Step>
![backup-restore-2](/docs/backup-restore-2.png)

백업된 목록을 확인할 수 있습니다.
</Step>
</Steps>


## 복원

백업된 목록에서 원하는 백업을 선택하고 복원 버튼을 클릭합니다.

<Callout type="info">
    복원시 확실한 복원을 위해 소피아를 껐다 켜주세요.
</Callout>
