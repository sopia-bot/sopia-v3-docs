## 소개

SOPIA <span className='text-fd-muted-foreground text-sm'>(소피아)</span>는 **방송용 매니저 프로그램**으로, <a href="https://www.spooncast.net/" className='text-[#FF4100]' target="_blank">스푼라디오</a>방송에 사용되는 프로그램입니다.
<br/>방송의 반복적인 작업을 줄여주고, 다양한 컨텐츠를 추가할 수 있습니다.
